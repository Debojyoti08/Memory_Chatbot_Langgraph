LangGraph FastAPI Chatbot with Persistent Memory

A simple chatbot API built with FastAPI, LangGraph, Ollama, and PostgreSQL.

This project demonstrates how to build a chatbot with:

Persistent conversation memory
Thread-based chat sessions
Tool calling support
FastAPI REST API
PostgreSQL checkpoint storage

Features
Create new chat threads
Continue previous conversations
Persistent memory using PostgreSQL
Retrieve conversation history
List all active threads
Tool execution support

Tech Stack
FastAPI
LangGraph
LangChain
Ollama
PostgreSQL
Pydantic
